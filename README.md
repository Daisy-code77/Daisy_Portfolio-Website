# ✨ Daisy Portfolio Website

A modern, high-performance personal portfolio built with **React 19**, **Vite**, and **Tailwind CSS**. Featuring a sleek dark-mode aesthetic, smooth animations with **Framer Motion**, and a fully responsive layout.

## 🚀 Key Features

- 💎 **Premium Aesthetic**: Modern glassmorphism design with a deep, elegant dark theme.
- ✨ **Fluid Animations**: Smooth entrance animations and scroll reveals powered by Framer Motion.
- 🌌 **Dynamic Background**: Interactive and visually engaging background elements.
- 📱 **Fully Responsive**: Optimized for all screen sizes—from high-res monitors to mobile devices.
- ⚡ **Lightning Fast**: Built with Vite for near-instant load times and HMR development.
- 🧩 **Modular Architecture**: Clean, component-based structure for easy maintenance and scaling.

---

## 🛠️ Tech Stack

- **Frontend**: React 19 (Hooks, Functional Components)
- **Styling**: Tailwind CSS (Custom Color Tokens, Responsive Utilities)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Build Tool**: Vite
- **Deployment**: Optimized for GitHub Pages / Vercel / Netlify

## 🏃 Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Daisy-code77/Daisy_Portfolio-Website.git
   ```

2. **Navigate to the directory**:
   ```bash
   cd "Daisy portfolio website"
   ```

3. **Install dependencies**:
   ```bash
   npm install
   ```

4. **Start the development server**:
   ```bash
   npm run dev
   ```

---

## 🌐 Deployment

To create a production build:

```bash
npm run build
```

The optimized files will be available in the `dist` folder, ready for deployment to any static hosting provider.
